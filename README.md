# Concurrency Examples in C++

Repository of concurrency related examples in modern C++17/20

## Build

### Prepare
```
$ git clone https://github.com/hnaik/concurrency_cpp.git
$ cd concurrency_cpp
$ mkdir build
$ cd build
```

### To Use Ninja

```
$ cmake -GNinja ..
$ ninja
```

### To Use GNU Make
```
$ cmake ..
$ make -j
```
